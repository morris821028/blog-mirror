UVa 751 - Triangle War

Share Comments
  • dp
  • 博弈

Problem

兩名玩家進行放邊的操作，當其中一名玩家成功創建一個三角形，則可以再放入一邊，直到該操作無法創建一個新的三角形，則進行輪替。最大化自己與對方的分數差異。

一開始先模擬兩名玩家的操作過程。

Sample Input

Sample Output

Solution

這題很妙，總之先定義狀態 dp(盤面狀況，剩餘三角形) = 最大化差異。

那麼轉移方程要看操作是否造成輪替，如果仍然為自己，則把下一步的最大化結果累加上來，反之扣除。
