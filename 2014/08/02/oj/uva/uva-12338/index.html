UVa 12338 - Anti-Rhyme Pairs

Share Comments
  • LCA
  • targan
  • trie

Problem

給一堆單詞，問任意兩個單詞的最常共同前綴長為何。

Sample Input

Output for Sample Input

Solution

對每一個單詞插入到 Trie 中，詢問任兩個字串的最長共同前綴時，相當於詢問兩個節點之間的最小共同祖先距離根多遠。

解決 LCA 問題，這裡採用離線 tarjan 做法。
