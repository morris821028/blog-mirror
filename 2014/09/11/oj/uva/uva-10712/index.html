UVa 10712 - Count the Numbers

Share Comments

  • dp
  • trie

Problem

找一個數字 M 介於 [A, B] 之間，且中間的 substring 包含 N 的個數為何。

Sample Input

Sample Output

Solution

建立一個 AC 自動機，使用傳統的 AC 自動機 dp，每一個節點當作一般 AC 自動機的走訪，並且猜測下一步的所有匹配符號。

為了要卡住上限，增加經過的狀態是否一直為前幾次臨界上限，若一直在上限，則搜索範圍將會被限制。
