Problem

有 N 種不同顏色的鈕扣，每一種燈泡分別有 $ n_{i} $ 個，請問拿 M 個鈕扣獲得的組合類型有多少種。

Sample Input

Sample Output

Solution

組合問題，相當於計算以下式子的整數解。

$$x_{1} + x_{2} + x_{3} + .. + x_{n} = M \\ 0 \le x_{i} \le n_{i}$$

事實上，利用 dp[i][j] 表示考慮前 i 種湊出 j 個組合數，得到 dp[i][j + k] += dp[i - 1][j] * C[j + 1 + k][k] 加入新種類的鈕扣。
