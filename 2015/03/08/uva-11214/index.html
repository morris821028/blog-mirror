Problem

用最少的皇后，覆蓋盤面中所有的 ‘X’。皇后之間的相互攻擊可以忽略不理。

Sample Input

Sample Output

Solution

直接套 DLX 最小重複覆蓋的模板，但是單純使用會狂 TLE。額外加上初始貪心，找到搜索的第一把剪枝條件，這一個貪心使用每次找盤面上能覆蓋最多 ‘X’ 的位置，進行放置皇后的工作。
